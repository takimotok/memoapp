# memoapp

For learning React Native.

## Tool set

- yarn v4

```sh
$ yarn -v
4.4.0
```

## Prerequisites

- docker compose cli > v2.26.x

```sh
$ docker compose version
Docker Compose version v2.26.1-desktop.1
```

## Usage

```sh
$ docker-compose up -d

$ docker-compose exec app /bin/bash

# inside docker container
$ yarn install
```
